### Como executar

Entre na **pasta do back** e instale as dependências
```
npm install
```

Entre na **pasta do front** e instale as dependências
```
npm install
```

Suba seu **servidor de back** (ele roda na porta **3000**)
```
npm start
```

Execute o **script.sql** no seu terminal ou workbench
